# Image Classification Using CNN

## Overview

This project implements an image classification model using a Convolutional Neural Network (CNN). The model is trained on the Fashion-MNIST dataset to classify clothing images into ten categories.

## Dataset

Dataset Name: Fashion-MNIST

Dataset Link:
https://github.com/zalandoresearch/fashion-mnist

## Technologies Used

* Python
* TensorFlow
* Keras
* NumPy
* Matplotlib
* Scikit-learn
* Seaborn

## Features

* Image preprocessing
* CNN architecture
* Model training and validation
* Accuracy and loss visualization
* Confusion matrix
* Classification report
* Model saving and loading

## Results

The CNN model successfully classifies clothing images into ten different categories with high accuracy.
