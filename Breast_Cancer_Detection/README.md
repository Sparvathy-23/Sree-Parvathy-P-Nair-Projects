# Breast Cancer Detection Using CNN

## Overview

This project implements breast cancer image classification using a Convolutional Neural Network (CNN). The model is trained on the BreastMNIST dataset to classify breast medical images.

## Dataset

Dataset Name: BreastMNIST

Dataset Link:
https://medmnist.com/

## Technologies Used

* Python
* TensorFlow
* Keras
* NumPy
* Matplotlib
* Scikit-learn
* Seaborn
* MedMNIST

## Features

* Medical image preprocessing
* CNN model implementation
* Model training and testing
* Accuracy and loss visualization
* Confusion matrix
* Classification report
* Model saving and loading

## Results

The CNN model successfully classifies breast medical images with strong performance on the testing dataset.
