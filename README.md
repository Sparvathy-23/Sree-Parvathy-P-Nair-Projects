# CNN Projects Using TensorFlow

## Overview

This repository contains three Convolutional Neural Network (CNN) projects developed using TensorFlow and Keras. Each project demonstrates the application of deep learning to solve a different image classification problem.

## Projects

### 1. Image Classification

* Dataset: Fashion-MNIST
* Objective: Classify clothing images into one of ten categories using a CNN.

### 2. Face Recognition

* Dataset: Labeled Faces in the Wild (LFW)
* Objective: Recognize individuals from facial images using a CNN.

### 3. Breast Cancer Detection

* Dataset: BreastMNIST
* Objective: Classify breast medical images using a CNN.

## Technologies Used

* Python
* TensorFlow
* Keras
* NumPy
* Matplotlib
* Scikit-learn
* Seaborn
* MedMNIST
